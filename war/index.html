Welcome to the ACME Telesales Application (GAE/J with Salesforce.com toolkit and Spring framework)

This application was built on Google App Engine using the new Force.com for Google App Engine Java Toolkit and the Spring framework.

The first version was done in Python and integrates with a Salesforce.com Developer org. I also did a Java version that runs off of BigTable.
